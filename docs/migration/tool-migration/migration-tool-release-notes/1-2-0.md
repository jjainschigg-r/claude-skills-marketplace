# Migration Tool 1.2.0

**Release Date:** 23 September 2025

## Enhancements

- [MSRH-431] Implemented the `utils/run-migration-replication-rules.sh`
  script which allows triggering migration replication rules in a sequence.
- [MSRH-482] Added the new configuration options
  `EVENT_BASED_PUSH_MIRRORING_REPLICATION_TRIGGER` and
  `EVENT_BASED_POLL_MIRRORING_REPLICATION_TRIGGER` to support Event Based
  trigger in migrated replication rules.

## Addressed Issues

- [MSRH-347] Fixed an issue wherein migration did not occur whenever
  `team-level` permissions were not defined.
- [MSRH-483] Fixed an issue wherein non-unique push and pull mirroring policies
  were not migrated.
